These are notes of some simple experiments ran for programming stochastic Kuramoto oscillators to perform computing. 
Based on ideas realeased by Unconventional, we extend it beyond trajectory based tasks to manipulating and generating 
empirical probability distributions of different shapes.

Note: the notebook is large containing many figures and animations of the results, and have been uploaded using git LFS.
So direct downloads will lead to errors. Use 'git clone https://github.com/entropich3atdeath/stochastic_oscillator_models.git' 
from your terminal to download the files and be able to view the notebook and the figures properly.
